# @formbox/docs

Docs and landing site for Formbox Renderer. Smart Web Messaging lives in the separate `@formbox/swm` package.

## Run locally

```bash
pnpm --filter @formbox/docs dev
```

Build:

```bash
pnpm --filter @formbox/docs build
```

Preview the landing build:

```bash
pnpm --filter @formbox/docs preview
```

## Output paths

- Docs site: `dist/`
